Kautilya — AI Tutor Bot
Kautilya is an AI-powered tutor bot that helps students solve academic queries. It provides relevant solutions, generates quiz questions, and maintains scores to track student performance. The application is built using Python and Streamlit.

Features
-AI-based academic query solving
-Quiz generation
-Score tracking
-Interactive user interface
-Fast and easy to use

Tech Stack
-Python
-Streamlit
-OpenAI API
