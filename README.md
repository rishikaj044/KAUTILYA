🚀 LearnNova — AI-Powered Personalized Tutor
LearnNova is an AI-driven intelligent tutoring system designed to personalize education and improve student mastery. It provides explainable step-by-step guidance, adaptive quizzes, multilingual support, and performance tracking — all in one unified learning platform.
Built for the future of AI in Education & Skilling.

To make high-quality, personalized, and confidence-building education accessible to every learner through explainable and inclusive AI.

🏗️ System Architecture Overview
LearnNova follows a modular AI-driven architecture:
Frontend (Streamlit UI)
 → Backend API Layer
 → AI Model Integration (LLM + Embeddings)
 → Vector Database for document retrieval
 → User Progress & Analytics Database
This ensures scalability, fast inference, and personalized feedback.
